# NLP Project

A project covering three core NLP tasks built from scratch:
- Text Classification
- Sentiment Analysis
- Named Entity Recognition (NER)

## setup

1. Clone the repo
2. Create a virtual environment: `python -m venv .venv'
3. Activate it: `.venv\Scripts\Activate
4. Install dependencies: `pip install -r requirements.txt`

## Tasks

### Text Classification
Classifying text into predefined categories using scikit-learn and transformers.

### Sentiment Analysis
Detecting positive, negative, or neutral sentiment from text.

### Tools and Libraries
- Python, NLTK, spaCy, HuggingFace Transformers
- scikit-learn, PyTorch
